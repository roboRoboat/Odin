CSS Selectors

CSS Selectors refer to the HTML elements to which the CSS rules will be applied.

    Selector Types
  • Universal Selector
    Selects all elements of all types. Represented by asterisk
    * {
      color:purple;
    }
  • Type Selectors
    Selects all elements of the specified HTML Element type:
    div {
      margin: 20px;
    }
  • Class Selectors
    Selects all elements of the specified HTML Element Class, uses '.':
    .warning-text {
      color: red;
    }
  • ID Selectors
    Selects the elements with the specified HTML ID, uses '#':
    #earning-report {
      display: inline-block;
    }
  • Grouping Selectors
    Used for groups of elements that share styles, comma separated list:
    .money,
    .grass {
      color: green;
    }
  • Chaining Selectors
    Used for selecting elements that have both both declarations. Just stick them together. Can't chain a type tag to another type tag
    #earning-report.col1 {
      color: blue;
    }
  • Descendant Combinator
    Used as relational selector between elements. Parent/Child relationship. Space separated values
    .ancenstor .child {
      font-size: 10px;
    }     blue
