The Box Model

  • Everything in the page is basically in a little box.
  • Box-sizing
  • W3 Link for Box-Sizing
  • From inside to outside, the order of the box-model properties is: Padding -> Margin -> Border
  • The "Box-sizing" property sets how the sizing of a box is calculated. By default it will include margins and padding. To remove that and get a more absolute size of box, set "box-sizing: border-box;"
  • To create more space between two elements, you should use the "margin" property.
  • To create more space between the contents of an element and it's border, you should use the "padding" property.
  • If you want two elements to overlap, you should use the "margin" property.

Display Property: Inline and Block

  • Normal Flow
  • Block vs. Inline Elements
  • Block
    Block elements will appear on the page stacked on top of one another. Each new element after a block element will start on a new line.
  • Inline
    Inline elements will go along with the surrounding flow. In the middle of an paragraph, an inline element will act the same as the rest of the p. Margins and padding should usually not be fiddled with on inline elements.
  • Inline-Block
    Behaves like a block elements (allows dimensions, margins, etc) but does not break after end of element.
  • Div and Span Elements
  • Div
    Divs divide the page into blocks
  • Span
    Groups text and other inline lements so they can be styled
